# 斗罗大陆 · 命运之书

基于斗罗大陆世界观的同人互动小说转盘游戏。

## 技术栈

- **UniApp** (Vue 3 + Vite)
- **Pinia** 状态管理
- **SCSS** 样式

## 快速启动

```bash
npm install
npm run dev:h5
```

或双击 `启动.bat`

浏览器访问 `http://localhost:3002`

## 项目结构

```
src/
├── components/     # UI 组件（转盘/叙事框/稀有度特效等）
├── config/         # 事件池/颜色/战力/NPC数据
├── pages/          # 页面（首页/转盘/结果/角色详情）
├── stores/         # Pinia 状态（角色/进度/世界）
├── utils/          # 工具（Canvas绘制/动画/战斗系统/存储）
├── App.vue         # 根组件
├── main.js         # 入口
└── pages.json      # 页面路由
```

## 核心系统

| 系统 | 说明 |
|------|------|
| 武魂 | 30+ 种，品质决定战力系数 |
| 魂环 | 10 槽位，十年→百万年 |
| 魂骨 | 7 部位，SSS 级品质 |
| 领域 | 杀神/蓝银/海神/天使/修罗/死亡 |
| 神位 | 海神/修罗神/天使神 |
| 武器 | 10 件，战力叠加 |
| 九考 | 海神/天使/修罗各 9 步 |
| 战斗 | 战力动态调整概率 |

## 事件数量

270+ 事件，覆盖 Lv.1-100 全等级。
